Instructions

Google Chrome is strongly recommended as the preferred internet browser for this application.

  • Click File → Print to open the print dialog menu, or simply type Ctrl+P (Windows/Linux) or ⌘+P (macOS).
  • After selecting your preferred printer and choosing your printer settings, the worksheet may be ready to print if all of the problems and the answer key is contained on one page. If so click "Print" and disregard the remainder of these instructions.
  • If the encrypted message is long (e.g. ≥35 characters), then the worksheet may split into two pages. This can be resolved by using Chrome's Scaling feature in the Print dialog menu before printing.
  • Click "More Settings", which expands more printer options. In the Scale dropdown, change "Default" to "Custom", and choose the largest number you can find to 100% that can contain all of the problems and answer key in one page. Refer to the image provided as an example.
