Math Decoder

a cryptographic mathematical game for children

Instructions: This application allows users to provide secret messages which are then encrypted into elementary mathematical equations based on grade level. In order to decrypt the encrypted message, the child secret agent decoder must solve the provided mathematical expressions to unlock the corresponding letter. Secret agent decoders then can piece together the message one letter at a time to complete the decryption process. Solve all of the math problems and be a cryptographic hero!
